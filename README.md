# AI Threat Intelligence Dashboard
A modern cybersecurity dashboard that performs real-time IP reputation lookup, malware threat feed monitoring, and global threat geo-visualization using live OSINT threat intelligence sources.
Built with Python, Streamlit, Plotly, Docker, and real external APIs (AbuseIPDB + IPInfo).

---

## Features
- IP Reputation Lookup
- Threat Intelligence Feeds
- Geo Heatmap (Plotly)

---

## Tech Stack
- Streamlit
- Docker
- AbuseIPDB, IPInfo
- Plotly

---
